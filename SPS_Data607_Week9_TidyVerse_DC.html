The dataset df_imf_rate is first transformed from a wide format into a long format using the pivot_longer() function. All columns except Currency are consolidated into a single variable, IMF_Date, while the corresponding values are stored in the C_Rate column. This restructuring facilitates time-series analysis by standardizing the data structure into a key-value format.

Following the reshaping step, the IMF_Date variable is cleaned by removing the leading “X” prefix introduced during the CSV import process. The cleaned values are then converted into proper Date objects using the format “%d.%b.%y”, ensuring consistent temporal formatting for downstream analysis and visualization.

The visualization is constructed using ggplot2 to compare the exchange rate trends between the Chinese yuan and Japanese yen over time. The dataset is first filtered to include only observations for these two currencies, while removing any missing values using drop_na() to ensure data integrity.

The filtered dataset is then mapped with IMF_Date as the temporal axis and C_Rate as the exchange rate value. Currency is used as the grouping and color aesthetic to enable clear differentiation between the two time series.

A line geometry (geom_line()) is applied to visualize the trend over time, providing a continuous representation of exchange rate movements. The plot is further refined with descriptive axis labels and a title indicating the time period of analysis (January 2026). Finally, theme_minimal() is used to enhance readability, and the x-axis text is rotated by 45 degrees to improve label visibility.
